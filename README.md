# Домашнее задание к лабораторной работе 8
## Условие
Вычислите для целого N , заданного пользователем вещественное значение N^2 + (N + 1)^2 + (N + 2)^2 + ... + (2 * N)^2
## Алгоритм
1. Задаем исходные данные
  - N — целое число, которое вводит пользователь.
  - i — счётчик цикла.
  - sum — переменная для накопления суммы квадратов чисел, инициализированная нулём
2. Приглашение к вводу пользвателем числа N.
3. Задаем цикл for
  for (i = N; i <= 2 * N; i++) {
		sum += (double)(i * i)}
4. Запускаем цикл.
5. Выводим результат sum
6. Конец
